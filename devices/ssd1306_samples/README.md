# SSD1306 Sample Images

This directory contains sample bitmap images for display on SSD1306 OLED displays.

## Image Requirements

- **Format**: PNG, JPG, or BMP
- **Recommended size**: 128x64 pixels (or 128x32 for smaller displays)
- **Color**: Any (will be converted to monochrome)
- **Aspect ratio**: 2:1 for 128x64, 4:1 for 128x32

## Adding Images

Simply place image files in this directory. They will appear in the image browser in the Device Panel.

## Sample Images Included

- `logo.png` - Example logo (placeholder)
- `icon.png` - Example icon (placeholder)
- `pattern.png` - Example pattern (placeholder)

You can replace these with your own images!
